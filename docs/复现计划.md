### 阶段一：环境搭建

**目标**：搭建可用的 conda 环境并安装所有依赖

**工作内容**：
- 创建 conda 环境（Python 3.10）
- 安装 CUDA 工具链（如需要）
- 安装项目依赖
- 配置环境变量
- 配置 WandB（可选）

**验证标准**：
- [ ] conda 环境 `lewm` 可正常激活
- [ ] `stable-worldmodel` 可成功导入
- [ ] `STABLEWM_HOME` 环境变量已设置
- [ ] `nvidia-smi` 显示 GPU 可用

**详细步骤**：参见 [复现步骤 - 环境搭建](复现步骤.md#一环境搭建)

## 常见问题排查

### CUDA Out of Memory (OOM)

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| 训练中断，报 OOM | Batch size 太大 | 减小 batch_size 到 32 或 24 |
| 训练中断，报 OOM | 嵌入维度太大 | 降低 embed_dim 到 128 |
| 训练中断，报 OOM | 历史长度太长 | 减小 history_size 到 2 |

### 训练速度慢

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| 每个 epoch > 20 分钟 | num_workers 太少 | 增加到 6-8 |
| 每个 epoch > 20 分钟 | 预取不足 | 增加 prefetch_factor |
| 每个 epoch > 20 分钟 | 精度不是 bf16 | 检查 precision 设置 |

### 损失不下降/NaN

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| 损失保持高位 | 学习率太低 | 尝试增加到 1e-4 |
| 损失变为 NaN | 学习率太高 | 降低到 1e-5 |
| 损失变为 NaN | 梯度爆炸 | 增加 gradient_clip_val |
| 损失波动剧烈 | batch size 太小 | 增加到 64 或使用累积 |

### 检查点问题

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| 检查点未保存 | 路径权限问题 | 检查 `$STABLEWM_HOME` 权限 |
| 加载检查点失败 | 文件路径错误 | 确认相对路径格式 |
| 加载检查点失败 | 格式不匹配 | 转换为 object.ckpt 格式 |

### 数据加载问题

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| KeyError | 列名不匹配 | 检查 keys_to_load |
| 数据加载慢 | num_workers 太少 | 增加工作进程数 |
| 数据加载卡住 | 缓存未启用 | 检查 keys_to_cache |

## 配置参考

### 双 GPU 推荐配置

```yaml
# config/train/lewm_dual.yaml
trainer:
  max_epochs: 100
  devices: 2
  accelerator: gpu
  precision: bf16
  gradient_clip_val: 1.0
  strategy: ddp

loader:
  batch_size: 32
  num_workers: 4
  persistent_workers: True
  prefetch_factor: 2

optimizer:
  type: AdamW
  lr: 5e-5
  weight_decay: 1e-3

loss:
  sigreg:
    weight: 0.09
```

### 单 GPU 备用配置

```yaml
# config/train/lewm_single.yaml
trainer:
  max_epochs: 100
  devices: 1
  accelerator: gpu
  precision: bf16
  gradient_clip_val: 1.0
  accumulate_grad_batches: 2

loader:
  batch_size: 48
  num_workers: 4
```

### 快速测试配置

```yaml
# config/train/lewm_debug.yaml
trainer:
  max_epochs: 2
  devices: 1
  limit_train_batches: 10
  limit_val_batches: 5

loader:
  batch_size: 16
  num_workers: 2

wandb:
  enabled: false
```

## 附录

### 快速命令参考

```bash
# 激活环境
conda activate lewm

# 单 GPU 训练
CUDA_VISIBLE_DEVICES=0 python train.py

# 双 GPU 训练
python train.py

# 评估

# 查看检查点
ls -lh $STABLEWM_HOME/checkpoints/

# 查看 GPU 状态
watch -n 1 nvidia-smi

# 清理缓存
conda clean --all
pip cache purge
```
